# Heart Disease Prediction using Machine Learning

Project Overview:
This project predicts whether a patient has heart disease using machine learning algorithms. The dataset contains patient health information such as age, blood pressure, cholesterol, chest pain type, ECG results, and other medical features.

# Objectives:
- Perform Exploratory Data Analysis (EDA)
- Preprocess and clean the dataset
- Train multiple machine learning models
- Compare model performance
- Select the best model for heart disease prediction

# Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

# Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

# Model Performance

| Model | Accuracy |
|--------|----------|
| Logistic Regression | 80.56% |
| Decision Tree | 77.78% |
| Random Forest | 80.56% |
| KNN | 52.78% |

# Best Model
Random Forest achieved one of the highest accuracies (80.56%) and was selected as the final model for this project.
